# GymNotes Legal Pages

This repository hosts GymNotes legal pages using GitHub Pages.

## Pages

- /privacy/
- /terms/

## Deploy

1. Push this repository to GitHub.
2. Open repository Settings.
3. Go to Pages.
4. Source: Deploy from a branch.
5. Branch: main, folder: /(root).
6. Save.

Live URLs:

- https://YOUR_GITHUB_USERNAME.github.io/gymnotes-legal/privacy/
- https://YOUR_GITHUB_USERNAME.github.io/gymnotes-legal/terms/
